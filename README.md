# POS-Pharmacy
 💊 POS Pharmacy System (2020)

Desktop Application – Visual Studio (C# / .NET)

During the COVID-19 pandemic in 2020, shortly after graduating from college, I developed a Point-of-Sale (POS) Pharmacy System using Visual Studio. This project was created to strengthen my software development skills and simulate real-world pharmacy operations in a digital environment.

🚀 Project Overview

The system was designed to help pharmacy staff efficiently manage daily transactions, inventory, and sales reporting. It focuses on speed, accuracy, and ease of use—essential for healthcare-related businesses.

🧩 Key Features
💰 Point-of-Sale (POS) transaction system
📦 Inventory and stock management (medicine tracking)
👨‍⚕️ Customer and product records management
🧾 Automated receipt generation
📊 Sales reports and daily summaries
🔐 Basic user authentication (admin/staff roles)
🛠️ Technologies Used
C# (.NET Framework)
Windows Forms (WinForms)
Microsoft SQL Server / Local Database
Visual Studio
🎯 Purpose & Learning Outcome

This project helped me understand:

CRUD operations in real applications
Database design and management
UI/UX for desktop systems
Business workflow simulation (pharmacy operations)
Problem-solving under real-world constraints
📌 Note

Built during the pandemic period as a self-learning project after graduation, demonstrating initiative, discipline, and continuous learning in software development.
