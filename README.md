💊 POS Pharmacy System (2020)

Desktop Application – C# / .NET (WinForms) | SQL Server

During the COVID-19 pandemic in 2020, shortly after graduating from college, I developed a Point-of-Sale (POS) Pharmacy System using Visual Studio. This project was built as a self-learning initiative to strengthen my software development skills and simulate real-world pharmacy operations in a desktop environment.

🚀 Project Overview

The system is designed to streamline daily pharmacy operations by improving the efficiency and accuracy of transactions, inventory tracking, and sales reporting. It focuses on speed, usability, and reliable data management—key requirements for healthcare-related businesses.

🧩 Key Features
💰 Point-of-Sale (POS) transaction system
📦 Inventory and stock management (medicine tracking)
👨‍⚕️ Customer and product records management
🧾 Automated receipt generation
📊 Sales reports and daily summaries
🔐 User authentication (Admin / Staff roles)
🛠️ Technologies Used
C# (.NET Framework)
Windows Forms (WinForms)
Microsoft SQL Server / Local Database
Visual Studio
🎯 Purpose & Learning Outcomes

This project helped me gain practical experience in:

CRUD operations in real-world applications
Database design and management
Desktop UI/UX development
Simulating business workflows (pharmacy operations)
Problem-solving under real-world constraints
